这是一套博客系统，功能比较简单，喜欢的朋友，可以自己丰富和完善

# 技术栈
前端：html、css、vue全家桶、各种插件
后端：node.js、express
数据库：mongodb、mongoose

兼容主流浏览器以及主流移动设备
